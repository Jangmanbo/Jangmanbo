### Stereo Matching
개발 인원|1인
---|---|
개발 기간|2021.12
개발 언어|Python

**Github**
  * https://github.com/Jangmanbo/Stereo_Matching

**📌소개**
  * 2개의 translation image의 corresponding point를 찾아 다양한 방법론을 사용하여 depth image를 생성
  * patch size, cost degree 등의 설정값에 따라 최적의 depth image을 생성하는 것이 목표
